Javascript T5 Problem Solving

.map, .reduce and filter explained by Jaive

array.prototype.map

The map method runs through an initial element, and calls a function on each element of that array and creates a new array with result of each call pushed in as new elements of that new array

array.prototype.filter

The filter method goes through elements of an array and return these elements only if they meet the condition as set in the function. Basically it runs a test on each element and only calkls back elements that pass this test.

array.prototype.reduce

Sometimes the videos are easier to understand then reading text. This one was the hardest to understand till i watched some videos. The. reduces method runs through each element of an array and runs a call back function on values in these array, however, the call back is the cumulative value of the previous call back etc and so on until the value it returns is the cumulative value of all the callbacks.

These methods reduce the amount of code, especially in reducing the dependence on for loops, and return results quicker.
