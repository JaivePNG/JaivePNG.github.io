How Does Javascript Compare to HTML & CSS

I would like to state for the record that it is 10x Harder than HTML and CSS. This is because it is responsible for the interactivity on a website, it is more labour intensive and has more methods, classes and functions to follow.

HTML is basically the language that determines whether a document becomes a web document and how it is read on a browser.

CSS is the styling for the HTML, that gives HTML different properties and helps it respond to the screen size of different documents.

Javascript is a programming language. It allows us to do more complicated 'things' on a website, such as calculations, registrations and other features. This can't be done in a HTML document. Javascript programs run in the web browser environment and are executed on the local computer.

There are other different programming languages such as PHP and Perl that are also used to write web based programs, but these two run on the remote host of the website.

Control Flow

There is a specific, top down approach in which the statements, instructions, function calls or a program are executed. Its a order sequence, referred to as synchronous code in Javascript. The code executes according to the instructions/steps/sequence of instructions put down.

However, Javascript is not that simple. We also have asynchronous code, where we have to run callbacks and other instructions at various times. We use devices such as 'loops' which affect the control flow of the program

Real Life Example

When I brush my teeth, I rinse my brush, i gurgle some water, I apply toothpaste, i brush my teeth, I rinse my mouth, I wash my brush and i put everything away. Task completed. I leave the bathroom. This would be an example of control flow - there is simple sequence I follow, the outcome of which is my teeth nice and clean.

If I was to integrate a loop into my toothbrush sequence, I would say that it would calculate how many times I have brush my molars, incisors etc, if I have brushed them all at least 3 times, the program considers them brushed and tells me to keep going, if I haven't, it says, keep brushing them. I can leave the bathroom only when the code is satisfied and the results are true ( i.e. i have brushed my molars and incisors x number of times)

What is A Javascript Function

A function in javascript is a block of code designed to execute a particular task when it is called or invoked. The function syntax looks like this: function name(parameter1, parameter2, parameter3) { code to be executed } The code to be executed is placed inside the curly brackets.

Functions are useful because they speed up our other javascript codes. This is because we can reuse the functions many times in many different arguments to produce different results.
